Nuevo Cliente
Recurrente: el cliente paga bajo un ciclo fijo (mensualidad, semanal, quincenal o anual) — ideal para membresías, rentas o servicios contratados. Aleatorio: el cliente compra de forma ocasional, sin un ciclo de pago fijo, cada vez que desea adquirir un producto o servicio.
Configuración del pago recurrente
Si NO ha pagado todavía, el pago quedará pendiente desde HOY y aparecerá de una vez en Ventas → Clientes con pago recurrente. Si marcas esta casilla, se asume que ya pagó y la próxima fecha de cobro será el siguiente ciclo.
La próxima fecha de pago se calcula automáticamente y siempre respeta este día fijo del ciclo, aunque el cliente pague parcial en otra fecha.
